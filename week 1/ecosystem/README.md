 IoT Ecosystem Theory  
Sensor: Detects changes in ambient conditions or the state of a system, then forwards or processes this information as an electrical signal.
Transducer: Converts a physical stimulus into an electrical signal (connected to an ADC - Analog to Digital Converter).
Actuator: Executes action or manipulation based on processed commands.

 Key Performance Metrics
Accuracy vs. Precision: Accuracy is how close a reading is to the true value; precision is how repeatable the readings are.
Resolution:The smallest change in a physical quantity that a sensor can detect.
