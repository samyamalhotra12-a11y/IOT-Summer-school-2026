if-else checks multiple conditions and can handle ranges, logical operators, and complex 
expressions. 
switch-case compares one variable against fixed values and is usually faster and cleaner for 
many discrete options.  
In IoT, if-else can be used for sensor threshold checks (e.g., temperature > 30°C), while 
switch-case is useful for device modes such as OFF, ON, SLEEP, and ALERT states.
