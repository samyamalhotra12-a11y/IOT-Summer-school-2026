ESP32 Wireless Capabilities

1. Wi-Fi: Cloud Connectivity
STA Mode (Station Mode):ESP32 connects directly to an existing local Wi-Fi router.
AP Mode (Access Point Mode): ESP32 acts as its own hotspot for direct configurations.
IP Stack: Features an integrated TCP/IP stack supporting protocols like HTTP, MQTT, and CoAP.
Over-the-Air (OTA): Enables remote firmware updates wirelessly without plugging into a computer.


2. BLE: Energy Efficient (Bluetooth Low Energy)
Low Power:Designed specifically for long-term battery-operated IoT nodes.
GATT Server: Organizes structure and data hierarchy into **Services** & **Characteristics**.
Advertising: Periodically broadcasts packets into the air for nearby devices to discover.
Local Interface:Best suited for Smartphone-to-ESP32 short-range setup and data exchange tasks.
